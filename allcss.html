Exemplos

Arredondando as bordas nas Imagens:

Para colocar Borda nas imagens usa-se:

img{
border-radius:20px; }

Colocando Fundo na imagem, colocando a imagem de fundo repetidas vezes, posicionando a imagem de fundo, colocar se a imagem vai mexer junto com a rolagem do mouse e colocar cor na imagem de fundo:

O código para essas ações de imagem de fundo são:

.testes{ background-image:url(
"/Projeto_html/imagens/seta.png");
background-repeat: repeat;
background-position: center top;
background-attachment: fixed;
background-color: purple;
text-align: center; }

Miniaturas das Imagens:

Para fazer miniaturas das imagens usa-se:

.teste3{ border: 2px solid #ddd;
border-radius: 3px;
padding: 100px;
width: 450px;
background-color: white;
text-align: center;
}

Rotação de imagem:

Para fazer miniaturas das imagens usa-se:

.teste5{ text-align: center;
-webkit-animation-name: keyFrameTeste5;
-webkit-animation-timing-function: ease-in-out;
-webkit-animation-iteration-count: infinite;
-webkit-animation-duration: 5s;
-webkit-animation-direction: alternate;
} }
