Trabalhando com imagens CSS

Com Cascading Style Sheets (CSS), você pode estilizar seu site e transformar as imagens relacionadas. Por exemplo, você pode criar um posicionamento estático ou fixo para os gráficos, definir planos de fundo e bordas, redimensionar e criar filtros interessantes para mostrar a arte.

CSS oferece inúmeras opções para essas tarefas. Gratificantemente, o CSS suporta muitos formatos e tipos de imagem, incluindo imagens raster (como JPEG), imagens vetoriais (como SVG), imagens combinadas (como ICO) e imagens sem arquivo (como gradientes CSS).

A transformação de imagem em CSS é baseada no <image> tipo de dados, que pode assumir a forma de elementos HTML, <url> ou <gradient> tipos de dados CSS, ou element() , Image() , cross-fade() ou image-set() funções. Depois de definir uma imagem, você pode transformá-la definindo propriedades CSS, conforme descrito neste artigo.
